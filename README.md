## ToSpecial

A SOCIAL NETWORK BASED WEBSITE FOR CONNECTING THE SPECIAL CHILDREN & THOSE WHO CARE ABOUT SPECIAL CHILDREN.

### Users:
- Family
- Doctor
- Educator
- Everyone

### Features
- Create account
- Posts filter
- Save posts
- Share posts
- Message options

### Technologies used
- UI(HTML,CSS)
- Backend(JS,PHP)
- Server(APACHE localhost)
